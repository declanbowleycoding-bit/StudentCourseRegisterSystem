Console line Java program, started April 6th, 2026.

Looping user menu: loops until valid input, executes option associated with input, sends you back to the menu unless menu exit was chosen, in which case the program ends. Can add students and courses, and view all students and courses.

Student object: stores id, name, and grade level of student. It has getters, setters, and a toString.

Course object: stores course code, course name, and max students. Includes getters, setters, and a toString.
